AGENTIC AI CODE EDITOR

Stop babysitting your AI. Give it a goal.

This is a real working agentic workspace: it calls the Google Gemini API live, edits real files in the browser, executes real JavaScript tests in a sandbox, computes a real diff, and opens a real pull request preview from whatever Gemini actually returns — nothing below is prerecorded.

How this is "real" vs a demo: Assigning a goal sends your project's files to the Gemini API and streams back an actual plan + actual code. Tests run in a real in-browser JavaScript sandbox (via a sandboxed Function), so pass/fail results and stack traces are genuine. Diffs are computed with a real line-diff algorithm over the before/after file content. The one thing a static, backend-less page genuinely can't do is spin up a real npm/OS shell — so the "terminal" executes real JS test code rather than shelling out to Node.

Note on the API key: the Gemini key is hardcoded in this file (GEMINI_API_KEY) for local, single-user use only. Don't deploy this file publicly or commit it to a public repo as-is — anyone who can view the page source can read the key.
